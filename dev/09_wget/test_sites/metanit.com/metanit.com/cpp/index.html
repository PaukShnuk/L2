О разделе

Данный раздел посвящен языкам программирования C и C++

C и C++ на сегодняшний день являются одними из самым популярных и распространенных языков программирования, который позволяют создавать приложения для любого спектра задач: разработка прикладных приложений, мобильная разработка и особенно системное программирование. Отличительной особенностью программ на C/C++ является высокая скорость работы, поэтому данные языки особенно часто используются в тех случаях, где необходимо обеспечить высокую производительность и быстродействие.

Для изучения базовых концепций языка Си в этом разделе имеется Руководство по Си.

Для тех, кто желает изучить язык C++, в разделе имеется Руководство по C++.
