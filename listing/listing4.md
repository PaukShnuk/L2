Что выведет программа? Объяснить вывод программы.

```go
package main
func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()
	for n := range ch {
		println(n)
	}
}
```

```
deadlock после отсчёта 0-9

    После 10 записей в канал горутина умрёт, а читающий из неё main, после 10 считываний,
будет ждать данных из канала и заблокируется.
