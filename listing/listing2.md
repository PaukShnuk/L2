Что выведет программа? Объяснить вывод программы. Объяснить как работают
defer’ы и порядок их вызовов.
```go
package main

import (
	"fmt"
)

func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}
func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}
func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

    Defer добавляет вызов дефер-функции в стек типа LIFO. Вызод defer при штатной 
работе происходит при вызовое return. Происхордит это всё следующим образом: данные, 
которые надо вернуть из функции записываются в область возврата, затем вызываются 
отложенные функции и, наконец, возврат значения туда, откуда была вызвана ф-я окружения.
    Область возврата в первом случае test() (x int) является именованной и к ней можно
обратиться, поэтому при Дефере мы обращаемся и успешно инкрементим х. Во втором случае
область возврата является анонимной и программа в Дефере меняет локальную переменную,
что никак не влияет на возвращаемое значение.
